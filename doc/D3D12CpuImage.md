# D3D12CpuImage

D3D12CpuImage is a raw CPU-side image buffer used as the DirectX/DXGI-only boundary for D3D12 texture transfer.

It is not an image-file abstraction.

## Scope

v1.2.0 supports single-plane, non-compressed formats.

Use higher-level libraries for PNG, JPEG, DDS, video, or media I/O.
