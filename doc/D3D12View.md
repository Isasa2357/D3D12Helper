# D3D12View

D3D12View provides descriptor description helpers for SRV, UAV, RTV, DSV, and CBV creation.

## Added in v1.7.0

- Texture2D SRV / UAV / RTV / DSV desc helpers
- Buffer SRV / UAV desc helpers
- Constant buffer view desc helper
- Descriptor handle validation helpers

These helpers create descriptor descriptions. Descriptor allocation and actual descriptor creation remain explicit.
