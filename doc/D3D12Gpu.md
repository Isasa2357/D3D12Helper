# D3D12Gpu

D3D12Gpu is the canonical module for resource, descriptor, upload, readback, transfer, binding, copy, resolve, mipmap, view, state, pipeline, shader compiler, and helper APIs.

## Added in v1.7.0

- D3D12View
- D3D12State

## Added in v1.6.0

- D3D12Copy
- D3D12Resolve
- D3D12Mipmap

## Added in v1.4.0

- D3D12DescriptorHeapSet
- D3D12BindingSet
- Root parameter binding helpers

## Added in v1.2.0

- D3D12CpuImage
- D3D12TextureTransfer

D3D12Framework remains available for v1.x compatibility.
